# 适配器

SpliceTree 核心引擎保持无头，适配器用于将核心数据与不同框架渲染层绑定。
所有适配器遵循以下原则：

- 保持核心引擎与 UI 完全解耦
- 支持所有插件系统
- 类型安全，框架无关

## 当前可用适配器

| 适配器        | 功能                            | 文档                     |
| ------------- | ------------------------------- | ------------------------ |
| Vue 3         | 将核心树绑定到 Vue 3 响应式系统 | [查看文档](adapters/vue) |
| React         | 将核心树绑定到 React Hooks      | 待补充                   |
| Svelte        | 将核心树绑定到 Svelte 响应式    | 待补充                   |
| WebComponents | 将核心树绑定到自定义元素        | 待补充                   |
